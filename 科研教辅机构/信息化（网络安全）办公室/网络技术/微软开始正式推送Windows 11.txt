微软开始正式推送Windows 11
今天，微软开始正式推送
Windows 11
。
“
让用户与之所爱，更近一步。”
这是微软日前官宣
Windows 11
正式推送日期时许下的愿景，并称
Windows 11
将给用户带来全新的体验。
实际上经过
Windows 11
预览版镜像泄露事件，再加上各种新闻爆料，下一代
Windows
的新鲜神秘感早已所剩无几。
而且每次提及
Windows
操作系统，让很多用户又爱又恨。“痛恨”
Windows
用户主要恼火其时常自动更新且
Bug
不断，该问题在
Windows 10
系统上显得极为突出，以至于有些用户到现在还怀恋
Win7
甚至
WinXP
系统。
还有微软的反复无常，此前曾随意抬高升级
Windows 11
的硬件门槛，导致很多提前享用
Windows 11
预览版用户，或将不得不回退到
Windows 10
版本。
话不多说，接下来让我们感受一下
Windows 11
正式版本与预览版本的不同之处吧。
迭代的
Windows 11
预览版本
自从
6
月
24
日
Windows 11
官宣以来，
Windows 11
预览版也在用户的测试中不断迭代更新。尽管
Windows 11
用全新
UI
的设计，史上最完美游戏体验，支持
Android
软件等新功能，深深吸引了用户，却依旧无法阻碍人们对
WIndows
系统的疯狂吐槽。
首先被用户吐槽最多的是微软严苛的升级计划。微软公司曾经发布升级
Windows 11
硬件要求，使得很多用户都卡在
TPM 2.0
，或者
CPU
版本不符合硬件需求等门槛上。即便用户使用虚拟机也不能逃脱这种严格的升级要求，以至于很多用户不得不放弃升级
Windows 11
，或者使用不被微软官方认可且提供更新支持的
Windows 11
预览系统。
除此之外，还有用户调侃
Windows 11 UI
设计上有苹果的身影。
Windows 11
界面呈现出所谓的毛玻璃效果，以及图标圆角化且自带阴影等特点，皆类似于
macOS
，甚至
Windows 11
的侧边滑出小组件，也有一丝
macOS
的味道。据悉苹果的
macOS 11
（
Big Sur
）侧边滑出小组件在右边，令人尴尬的是
Windows 11
在左边。
（来源于微软官网）
在其它方面，
Windows 11
预览方面还支持明亮模式和暗黑模式。据悉
Windows 11
在
UI
许多新设计，都来源于
Windows 10X
的设计思路，尽管后者最终被微软取消。
总的来说，尽管
Windows 11
预览版本存在一些不足，但是刚刚推送的
Windows 11
正式版本不仅修复一些
Bug
，还为我们带来一些“小惊喜”，下面让我们一起来欣赏吧。
使用全新开始菜单和任务栏
Windows 11
相比
Windows 10
最大的变化，就是其全新
UI
界面，原先用户最熟悉的开始菜单被重新设计—从左侧转移到中间。
而
Windows 11
正式版开始使用新的开始菜单和任务栏，从新系统的每一个声音、字体和图标，微软开发团队都深思熟虑地考虑了每一个像素和细节，希望为用户带来更现代、更新鲜、更美好的体验。
通过全新的开始菜单，用户可以快速访问自己关心的内容和应用程序，而且“开始”菜单利用云和
Microsoft 365
的强大功能，无论哪个平台或者设备上（即便是在
Android
或
iOS
设备），用户都可以在“开始”菜单上查看到最近文件。
（来源于微软官网）
全新的
Widgets
与
Microsoft Teams
微软方面介绍，为了让用户更接近自己关心的内容和信息，
Windows 11
正式版推出了全新
Widgets
，这是一种由
AI
和网络提供支持的全新个性化订阅源。
当小部件像一块玻璃一样在屏幕上滑动时，通过单击或从左侧滑动来打开用户的个性化提要。无论是通过
Outlook
日历和待办事项列表集成浏览用户的一天，了解最新的头条新闻或明天的天气预报，还是查看用户最喜爱的
OneDrive
照片记忆，
Widgets
小部件能让用户喜欢的一切都触手可及。
（来源于微软官网）
WIndows 11
正式版本中
Microsoft Teams
也给用户带来一种全新的体验。用户只需单击一下，就可以使用个人帐户与自己的朋友、家人和其他联系人进行聊天、语音或视频通话，无论他们使用的是哪种设备或平台——
Windows
、
Mac
、
Android
或
iOS
。
Microsoft Teams
不仅适用于拥有智能设备的人，还可以通过
SMS
文本消息加入对话。此外，用户也可以直接从
Chat
中打开
Microsoft Teams
，通过事件安排、快速投票、文件共享等功能实现自己的计划安排。
（来源于微软官网）
可激发创造力的生产力工具
Windows 11
不仅具有强大的用户新体验，还可提高用户的工作效率并激发用户的创造力。
从多个显示器和浏览器选项卡到在
Word
或
PowerPoint
中编辑文档，
Windows 11
都可轻松地在

上处理多项任务。在
Windows 11
正式版本中，
Windows 11
引入了全新的
Snap
布局、
Snap
组件和虚拟桌面功能，借助适用于更大屏幕的全新三栏布局，用户可以轻松地将所需的所有内容呈现在桌面上，从而帮助用户整理窗口和优化屏幕空间，尽量保证用户工作学习状态。
（来源于微软官网）
而且
Windows 11
在开发初始就考虑了一些残障人士的需求，并称
Windows 11
是有史以来最具包容性的
Windows
版本，操作与系统界面对残障人士极其友好。
Windows 11
提供了熟悉的辅助技术，例如讲述人、放大镜、隐藏式字幕和
Windows
语音识别，为不同类别的残障人士提供了支持和便利。
微软还改进了触摸体验，不仅在速度和效率方面得到提升，而且还优化了墨迹书写和语音体验。平板模式下，其页面布局没有完全改变，任务栏中的图标之间距离变大了，使用户更方便触摸图标、移动窗口或者调整图标大小。
（来源于微软官网）
Windows 11
正式版将延期支持安卓

微软称，为了实现了安卓

根据微软官方介绍，鉴于
