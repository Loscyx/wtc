在线学习慕课平台建设

学校和江苏金智教育信息技术有限公司合作，建设了武汉职业技术学院在线学习平台。平台基于
web2.0
、云计算、移动计算、社交网络等技术，为学习者提供更智能的学习体验和更优质的碎片化课程资源。平台整合了线上线下的学习行为，为学习者和教学者提供知识元学习，师生互动，单元检测，教学辅导等多种学习体验，无论是学生，社会学习者，企业员工都可以通过多终端在无边界的学习网络中畅游和交互。
在线学习平台设计遵循
J2EE
规范，支持跨操作系统使用，可以部署到
UNIX
、
LINUX
和
Windows
平台运行，应用系统的开发及运行结构要基于统一的技术开发平台的三层架构，即
Web
服务器、应用支撑服务器和数据库服务器，支持
SOA
。
在线学习平台提供手机客户端和平板客户端供教师和学生用户使用，手机客户端和平板客户端都支持主流的
Android
、
iOS
移动操作系统，满足移动学习、泛在学习的需求。
在线学习平台
UI
采用
Web2.0
用户界面风格，系统采用
Web2.0
设计规范，提供体验良好的互联网风格的
UI
，方便教师和学习者的操作。
在线学习平台采用采用了
redis
缓存、
CDN
视频加速、
ecs
云计算单元、
NOSQl
数据库等云计算技术保证满足
100
万用户以上，千量级用户并发运行，系统能及时的响应用户操作请求，并能平滑升级以支持平台用户的增长。
系统遵循
SAAS
（
Software-as-a-Service
软件即服务）化的设计，支持多个教学单元在一个平台上可以拥有数据、内容互不干扰的平台使用环境，无需再次安装软件，即可完成本校的在线课程出版、网络学习环境搭建。
在线教学平台共包括：教师教学空间模块、学生学习空间模块、移动学习空间、督学及统计分析模块、考试中心模块、平台后台管理模块、校内资源库平台模块。利用平台可以全景展示学校课程建设、资源建设等教学成果；并为教师提供网络课程制作工具，快速开发高质量的公共网络课程。支持依托于网络的混合式多种学习模式，营造高智能化师生互动环境。
