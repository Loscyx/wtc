中南财经政法大学工商管理学院自考本科衔接教育毕业论文写作格式
来源：中南财经政法大学工商管理学院

（一）内容要求
题目与标题：题目一般应在20字以内，能简单、具体、确切地表达论文特定内容。题目应避免使用缩略词、首字母缩写字、字符、代号和公式。必要时可使用副题。
摘要：即内容提要，浓缩概括本设计报告的内容、方法和观点，以及取得的成果和结论。摘要应反映整个论文内容的精华，以300字内为宜。撰写摘要时要用精炼、概括的语言来表达，每项内容不宜展开论证或说明，要客观陈述，不加主观评价；成果和结论性字句是摘要的重点，文字叙述要多些，以加深读者的印象，要独立成文，选用词语要避免与全文尤其是前言和结论部分雷同。
在摘要部分下提供本篇设计报告的“关键词”，一般选择3~5个关键词，关键词总字数不超过20字。
目录：目录编排最多使用三级目录，二、三级目录相对前级目录右缩进一个汉字位置，根据纸张大小（一般为A4幅）在目录标题右边加上适当长度间断线后写页码，注意页码纵向右对齐。
正文：正文是文章的主体部分，必须进行详细的描述和说明，主要内容包括：问题的提出，工作的基本前提、假设和条件；方案的拟定；基本概念和理论基础；实验方法、内容及其分析；理论论证，理论在课题中的应用，课题得出的结论及对结果的讨论等。对于正文中图件用绘图墨水在透明纸上描绘或用计算机打印，图中标注符合国家规范要求。
结论：结论集中反映作者的成果，表达作者的见解，是全文的思想精髓，是论文价值的体现。结论包括对整个工作进行归纳和综合得出的总结，重点介绍课题研究取得的成果及存在的问题，还应提出进一步发展研究的见解与建议。
参考文献：参考文献是毕业设计不可缺少的组成部分，他反映毕业设计取材来源、材料的广博程度和可信程度。要求作者将设计中参考过的主要文献列出，以示对文献作者的尊重；让读者明晰论文中的观点或成果与前人工作的界限。
可以按序号、作者、书刊名、出版社、出版年月、第几版（期）顺序列出。
附件：对不适宜归纳在正文中具有独立内容的材料（如图表、计算报告、公式推导等）作为附件处理，附件应附名称，多个附件应有编号，格式与正文要求相同。
（二）毕业设计装订结构
（1）毕业论文封面
（2）学生毕业论文指导老师评阅表
（3）目录（按照正式出版物格式要求编排）
（4）正文
（5）结论
（6）参考文献
（7）附件
（三）结构编排、字体和图件规范要求
1、毕业设计题目及学生、指导教师姓名字体。
毕业设计题目：二号宋体加黑居中。
学生姓名、指导教师姓名：小四宋体加粗居中。
2、毕业设计分章、节、项三级标题，用阿拉伯数字加圆点表示，如：
第一章表示为：1. X X X X X采用小三号黑体加粗居中左顶格方式排；
第一章的第一节表示为： 1.1 X X X X X采用四号宋体加黑顶左空二字排；
第一章的第一节的第一项表示为： 1.1.1 X X X X X采用小四号仿宋加黑项左空二字排。在“项”标题下还细分，则用阿拉伯数字加小四号宋体括号表示。
3、毕业设计正文用A4纸幅面打印，纸的四周要留足空白边缘（上3.0 cm，下2.5 cm，左3.2cm，右2.2cm），便于在左边装订。
4、毕业设计正文采用“小四号宋体”字编排，每页标准方块汉字数850—950字（25-27行/页，每行32-34字），版面清晰，容易阅读。
5、毕业设计采用规范的汉语简化字。年份一概写全数（例：1998年不能写成98年）；分数、世纪、年代均以汉字表示（例：三分之二不能写成2/3，二十世纪九十年代不能写成20世纪90年代）。
6、正文中的插图一律采用计算机绘制。图名一般注于插图下边1.0 cm居中位置，采用“小四号宋体”加黑，文字说明采用“五号宋体”；插图中的注解采用“小五号宋体”。
7、毕业设计的大幅附图采用计算机绘制。
8、毕业设计中的资料数据表，其名称采用“小四号宋体”字加黑居表上中部，表的编码采用“五号宋体”字加黑位于表的左上角；表中采用“五号宋体”字。
9、毕业设计中的物理量、量纲及符号均采用国际标准（SI）和国家标准（GB）。
